# CiblOrgaSport - Frontend

**CiblOrgaSport** est une application web dédiée à la gestion et à l'organisation d'événements sportifs.

## 🚀 Démarrage rapide

### Prérequis
- Node.js (version 18 ou supérieure)
- Yarn (recommandé) ou npm

### Installation

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/Salas3108/CiblOrgaSport_Front.git
   cd CiblOrgaSport_Front
2. **Installer les dépendances**

 yarn install
# ou
 npm install

3. **lancer le project**

 yarn run dev
